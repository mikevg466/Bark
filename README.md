## To start
```
yarn install
createdb bark
yarn seed
create a secrets.js with keys for:
 * process.env.GOOGLE_CLIENT_ID
 * process.env.GOOGLE_CLIENT_SECRET
 * process.env.GOOGLE_CALLBACK
npm run start-dev
```

You should now be able to enjoy your very own pokeshoppe on http://localhost:8080/
